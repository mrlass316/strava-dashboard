# Strava Dashboard — Luis Sánchez

Dashboard personal de entrenamiento con datos de Strava y coaching inteligente.

**Stack:** React 18 + Recharts + Tailwind CSS (standalone HTML, sin build step)

**Features:**
- Overview con CTL/ATL/TSB, radar de atleta y volumen
- Coaching con TRIMP, eficiencia cardíaca y potencia
- Running con evolución de pace y scatter plots
- Fitness con zonas cardíacas y predicciones VO2max
- Tracking de tenis con proyección de reemplazo
- Coach AI con recomendaciones dinámicas en cada gráfica

**Deploy:** Conectado a Netlify — cada push a `main` despliega automáticamente.
